# Dolphinator
Arduino based ultrasonic distance sensor with vibration feedback for educatory puropse

## Parts list
- Arduino: Uno Rev.3
- Ultrasonic Sensor: HC-SR04
- Vibration motor: Grove - Vibration motor
- Grove cable
- 3-wire extension cable from Arduino to HC-SR04
- Red LEDs
- Standard Powerbank
- Bike helmet
- Cardboard box
